== Description ==

Displays the uncut title of your site on the admin screens. See the Screenshots section for details.

== Installation ==

1. Upload `full-site-title` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
